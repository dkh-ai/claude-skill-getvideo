# Self-Improvement Knowledge Base

Эта директория содержит накопленные знания агента.
Обновляется автоматически после каждой сессии.

## Структура
- `sessions/` — краткие отчёты о каждой сессии (что делал, что выучил)
- `patterns/` — выявленные паттерны и best practices для этого проекта
- `failures/` — зафиксированные ошибки и способы их избежать
- `decisions/` — архитектурные решения и их обоснования (ADR-формат)
- `metrics/` — количественные метрики качества (тесты, время, ошибки)
